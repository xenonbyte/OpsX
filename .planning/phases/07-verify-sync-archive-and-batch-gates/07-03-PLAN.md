## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| verified/synced change state -> archive move | unsafe preconditions or destination collisions can archive incomplete work or overwrite prior evidence |
| batch target set -> per-change loops | shared mutable state can leak blockers, task groups, or drift warnings between changes |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-07-05 | Tampering | `lib/batch.js` | mitigate | Load each change independently, aggregate plain result objects, and reserve global aborts for workspace/argument/runtime failures only. |
</threat_model>
